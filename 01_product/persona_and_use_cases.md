# Persona & Use Cases

## 주요 페르소나

### 현장 작업자
- 목표: 빠르게 문제를 해결하고 실수 없이 절차를 수행
- 고통: 매뉴얼 탐색 어려움, 전문가 연락 지연, 리포트 작성 부담
- 필요한 기능: 핸즈프리 음성 명령, AR 주석, 체크리스트, 자동 기록

### 원격 전문가
- 목표: 여러 현장을 효율적으로 지원하고 정확한 지시 제공
- 고통: 현장 상황 파악 부족, 사진/영상 품질 문제, 사후 기록 누락
- 필요한 기능: 실시간 영상, 확대/정지화면, 주석, 장비 이력, 세션 큐

### 운영 관리자
- 목표: MTTR 감소, 인력 효율화, 품질 표준화
- 고통: 작업 품질 편차, 고장 원인 데이터 부족, 교육 비용
- 필요한 기능: KPI 대시보드, 리포트 승인, 지식 활용 분석

### 보안/IT 관리자
- 목표: 데이터 유출 방지, 접근권한 통제, 감사 대응
- 고통: 현장 영상 민감정보, 외부 전문가 접근, 장치 분실
- 필요한 기능: SSO, RBAC, MDM, 암호화, 감사로그, 온프레미스

## 핵심 유스케이스
| ID | 유스케이스 | 설명 | 성공 기준 |
|---|---|---|---|
| UC-01 | 긴급 장애 복구 | 장비 오류 발생 시 원격 전문가 즉시 연결 | MTTR 20~40% 감소 목표 |
| UC-02 | 신입 작업자 절차 안내 | 전문가와 AI가 단계별 체크리스트 제공 | 재작업률 감소 |
| UC-03 | 벤더 원격 지원 | 외부 장비 제조사 전문가가 제한 권한으로 접속 | 출장비 절감, 보안 감사 가능 |
| UC-04 | 품질 검수 | 전문가가 원격으로 설치/수리 결과 확인 | 검수 리드타임 단축 |
| UC-05 | 교육 콘텐츠 생성 | 세션을 편집해 베스트 프랙티스 콘텐츠로 전환 | 교육 자료 자동 축적 |
| UC-06 | 규정 리포트 | 작업 증거와 승인 과정을 리포트로 자동 생성 | 감사 대응 시간 단축 |

## 고부가 시나리오
1. 장비 오류코드 인식 → 관련 매뉴얼 자동 호출
2. 영상 속 계기판/라벨 OCR → 부품번호와 절차 매칭
3. 전문가 주석 → AR 앵커로 고정 → 작업자 이동 후에도 위치 유지
4. 위험 문구 감지 → AI가 작업 중단/전문가 확인 요청
5. 세션 종료 → 리포트 초안 → 관리자 승인 → CMMS 업데이트
