# Competitive Positioning

## 시장에 이미 있는 기능
현재 시장에는 원격 영상 지원, AR 주석, 스마트글래스/모바일 지원, 세션 리포트 기능을 제공하는 솔루션이 존재한다. 따라서 단순히 “현장 화면 공유 + 원격 주석”만으로는 차별화가 약하다.

## 차별화 방향
| 기존 원격지원 | 제안 시스템 |
|---|---|
| 전문가가 화면을 보고 지시 | AI가 매뉴얼·이력·절차를 같이 준비 |
| 세션 중심 | 워크오더·자산·문서·리포트까지 연결 |
| 단순 녹화/캡처 | 증거 기반 자동 리포트와 감사 추적 |
| 일반 주석 | 장비/부품/절차 객체와 연결된 AR 주석 |
| 제품 기능 중심 | 산업별 운영 패키지와 지식팩 제공 |

## 포지셔닝 문장
“우리는 AR 원격지원 도구가 아니라, 현장 유지보수 업무를 AI와 XR로 표준화하고 자동화하는 엔터프라이즈 작업 운영 플랫폼이다.”

## 경쟁 우위 설계
1. **지식 통합:** 매뉴얼, SOP, 도면, 장비 이력, 과거 리포트를 통합 검색
2. **산업별 템플릿:** 플랜트, 병원, 통신, 건설, 선박/항공 특화 체크리스트
3. **보안 배포:** 온프레미스와 프라이빗 모델 지원
4. **리포트 자동화:** 작업 종료 후 실제 업무 산출물을 자동 생성
5. **현장 UX:** 음성 중심, 장갑 친화, 저대역폭, 스마트글래스 최적화
