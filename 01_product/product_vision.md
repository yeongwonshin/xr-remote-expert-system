# Product Vision

## 비전
전 세계 모든 현장 작업자가 숙련 전문가와 AI 지식을 즉시 호출해, 복잡한 장비 문제를 안전하고 빠르게 해결하는 산업용 XR 작업 운영체제를 만든다.

## 제품 원칙
1. **작업자 우선:** 화면을 많이 누르지 않고 음성·제스처·원격 주석 중심으로 작동한다.
2. **근거 기반 AI:** AI 답변은 항상 문서 출처, 버전, 페이지, 신뢰도를 함께 제시한다.
3. **전문가 통제:** AI는 작업을 보조하되, 위험 작업과 최종 판단은 전문가가 승인한다.
4. **증거 자동화:** 모든 중요한 순간은 리포트와 감사 로그로 남긴다.
5. **기업 통합:** 기존 시스템을 대체하기보다 연결한다.

## 제품 계층
| 계층 | 목적 | 예시 기능 |
|---|---|---|
| Connect | 현장과 전문가 연결 | 영상, 음성, 채팅, 다자 세션 |
| Guide | AR로 정확히 지시 | 화살표, 원, 텍스트, 3D 핀, 체크리스트 |
| Know | 지식 검색 및 추론 | RAG, 매뉴얼 검색, 장비 이력, 유사 고장 |
| Record | 작업 증적화 | 녹화, 스냅샷, 음성 전사, 주석 로그 |
| Automate | 후속 업무 자동화 | 리포트, 부품 요청, 티켓 업데이트 |
| Govern | 엔터프라이즈 통제 | RBAC, SSO, 감사, 데이터 정책 |

## 핵심 제품 경험
작업자는 장비 앞에서 “이 장비 모델 X의 오류코드 E37 해결 절차 보여줘”라고 말한다. AI는 해당 장비의 최신 매뉴얼과 과거 고장 이력을 찾아 요약하고, 안전 체크리스트를 먼저 제시한다. 원격 전문가는 작업자의 시야 위에 밸브 위치를 원으로 표시하고, AI는 세션 종료 후 원인·조치·부품·증거 이미지가 포함된 리포트를 생성한다.
