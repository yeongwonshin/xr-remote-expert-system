# Support & SLA

## 지원 등급
| 등급 | 대상 | 지원 시간 | 응답 목표 |
|---|---|---|---|
| Standard | 일반 SaaS 고객 | 영업시간 | P1 4시간 |
| Business | 대기업/다현장 | 12x5 | P1 1시간 |
| Enterprise | 기간산업/병원/통신 | 24x7 | P1 30분 |
| Mission Critical | 온프레미스/고보안 | 24x7 전담 | P1 15분 |

## 장애 등급
| 등급 | 정의 |
|---|---|
| P1 | 전체 세션 불가, 보안 사고, 감사로그 중단 |
| P2 | 특정 테넌트 핵심 기능 장애, AI/RAG 불가 |
| P3 | 일부 기능 지연/오류, 리포트 실패 |
| P4 | 사용 문의, 설정 지원, 개선 요청 |

## 고객 온보딩 지원
- 관리자 교육
- 현장 작업자 교육
- 전문가 콘솔 교육
- 지식 문서 수집/정제 지원
- 파일럿 KPI 정의
- 보안 검토 문서 제공

## 운영 리포트
매월 고객에게 다음을 제공한다.
- 세션 사용량
- 평균 세션 시간
- MTTR 변화
- 리포트 생성 수
- AI 답변 채택률
- 검색 실패 질문
- 문서 최신화 권고
- SLA 성과
