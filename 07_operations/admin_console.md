# Admin Console

## 주요 메뉴
1. 조직/테넌트 관리
2. 사용자/그룹/역할 관리
3. 현장/장비 관리
4. 지식 문서 관리
5. AI 정책 관리
6. 세션/리포트 관리
7. 보안/감사로그
8. 연동 관리
9. 사용량/비용 관리
10. 디바이스 관리

## 관리자 기능 상세
### 사용자 관리
- SSO 그룹 매핑
- 역할 부여
- 외부 벤더 초대와 만료일 설정
- 자격/교육 이수 상태 관리

### 장비 관리
- 장비 코드, 모델, 위치, 시리얼
- QR/NFC 태그 발급
- 관련 매뉴얼/SOP 연결
- 장비별 고장 이력 조회

### 지식 관리
- 문서 업로드
- 버전 승인
- 인덱싱 상태 확인
- 검색 품질 테스트
- 문서별 접근권한 설정

### AI 정책
- 산업별 위험 키워드
- 고위험 작업 승인 조건
- 모델/인덱스 선택
- 답변 언어
- 리포트 템플릿

### 감사/보안
- 세션 접속 이력
- 외부 초대 이력
- AI 질문/답변 이력
- 영상/리포트 다운로드 로그
- 정책 변경 이력

## 대시보드 카드
- 오늘의 세션 수
- 긴급 세션 대기열
- 평균 복구시간
- AI 답변 채택률
- 문서 인덱싱 실패
- 보안 경고
- SLA 위반 위험
