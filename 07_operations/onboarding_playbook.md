# Onboarding Playbook

## Phase 1. Discovery
- 대상 현장과 장비 선정
- 기존 워크오더/문서/리포트 프로세스 확인
- 네트워크 환경과 디바이스 제약 확인
- 보안/개인정보 요구사항 수집
- 파일럿 KPI 합의

## Phase 2. Knowledge Setup
- 장비별 매뉴얼/SOP/도면 수집
- 문서 버전 정리
- 오류코드/부품번호 메타데이터 태깅
- RAG 인덱스 생성
- 전문가와 Golden Q&A 검증

## Phase 3. System Configuration
- 조직/사용자/역할 설정
- SSO 연동
- 현장/장비 등록
- 리포트 템플릿 설정
- 외부 벤더 정책 설정

## Phase 4. Field Pilot
- 작업자/전문가 교육
- 5~10개 실제 작업 세션 수행
- 네트워크 품질과 UX 이슈 수집
- 리포트 품질 검토
- KPI 초기 측정

## Phase 5. Scale
- 장비군 확대
- 현장 확대
- CMMS/EAM 연동
- 운영 대시보드 정착
- 보안/컴플라이언스 검토 완료

## 파일럿 체크리스트
- [ ] 대상 장비 3종 이상 선정
- [ ] 매뉴얼/SOP 최신본 확보
- [ ] 전문가 3명 이상 배정
- [ ] 현장 작업자 10명 이하로 초기 시작
- [ ] 세션 녹화/보존 정책 합의
- [ ] 성공 KPI와 기준값 수집
- [ ] 리포트 승인자 지정
