# Pilot to Scale Plan

## 0단계: Pre-Sales Assessment
- 대상 장비와 장애 유형 선정
- 기존 평균 복구시간과 출장비 수집
- 문서 품질 점검
- 보안 요구사항 확인

## 1단계: Controlled Pilot
- 1개 현장, 3개 장비군
- 모바일 중심, 기본 AR 주석, RAG 검색, 리포트 자동화
- 성공 기준: 세션 성공률, 리포트 채택률, 작업자 만족도

## 2단계: Operational Pilot
- 2~3개 현장 확대
- 스마트글래스 일부 도입
- 워크오더/티켓 연동
- 관리자 대시보드 운영

## 3단계: Enterprise Rollout
- 전사 SSO/SCIM 연동
- 현장별 정책 배포
- 산업별 리포트 템플릿
- CMMS/EAM 양방향 동기화
- SLA 운영 계약

## 4단계: AI Optimization
- 유사 고장 추천
- 부품 요청 자동화
- 작업자 교육 콘텐츠 생성
- 예지보전/품질분석 연계

## 확장 의사결정 기준
- 파일럿 ROI가 양수인가?
- 현장 작업자가 실제로 계속 사용하는가?
- 전문가 지원 시간이 절감되는가?
- AI 답변이 안전하고 유용한가?
- IT/보안 요구사항을 만족하는가?
