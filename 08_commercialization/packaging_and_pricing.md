# Packaging & Pricing

## 패키지 구성
| 패키지 | 대상 | 포함 기능 |
|---|---|---|
| Starter | 단일 현장 PoC | 모바일 영상, 음성, 2D 주석, 기본 리포트 |
| Professional | 다현장 운영 | AR 주석, AI/RAG, 자동 리포트, 관리자 콘솔 |
| Enterprise | 대기업/보안 산업 | SSO, 감사로그, 연동, 권한정책, SLA |
| Private/On-Prem | 고보안 산업 | 전용 배포, 프라이빗 모델, 데이터 레지던시 |

## 과금 단위
- Technician Seat: 현장 작업자 계정
- Expert Seat: 원격 전문가 계정
- Admin Seat: 관리자 계정
- AI Copilot Add-on: RAG/요약/리포트 AI 기능
- Knowledge Pack: 문서 인덱싱 용량
- Media Storage: 녹화/스냅샷 보관 용량
- Integration Pack: CMMS/EAM/ERP 연동
- On-Prem License: 설치형 라이선스 + 유지보수

## 가격 논리
가격은 단순 좌석이 아니라 고객의 절감 가치에 연결한다.
- 전문가 출장비 절감
- 장비 다운타임 감소
- 리포트 작성 시간 감소
- 신입 교육 비용 감소
- 품질/안전 리스크 감소

## 기업 내부 구축 모델
| 항목 | 수익 |
|---|---|
| 초기 구축비 | 아키텍처 설계, 설치, SSO/연동, 지식 구축 |
| 연간 라이선스 | 사용자/현장/서버 기준 |
| 유지보수 | 장애 대응, 업그레이드, 보안 패치 |
| 전문 서비스 | 산업별 템플릿, 매뉴얼 정제, AI 평가 |

## 업셀 포인트
- 스마트글래스 대량 배포
- 온프레미스 AI/RAG
- 산업별 리포트 템플릿
- 고급 분석 대시보드
- 다국어 실시간 번역
- 부품/재고/구매 자동 연동
