# Device Support Matrix

## 1차 지원 디바이스
| 디바이스 | 장점 | 제약 | 권장 산업 |
|---|---|---|---|
| iOS/Android 스마트폰 | 도입 쉬움, 카메라 품질 높음 | 양손 작업 불편 | 모든 산업 PoC |
| 태블릿 | 큰 화면, 도면 보기 좋음 | 휴대성 낮음 | 건설, 병원, 플랜트 |
| HoloLens 계열 | 공간 앵커, 핸즈프리 | 비용, 착용성, 배터리 | 고부가 정비, 교육 |
| RealWear 계열 | 산업현장 내구성, 음성 조작 | 시야각/AR 표현 제한 | 플랜트, 에너지, 선박 |
| Android Smart Glass | 저비용 확장 가능 | 기기별 성능 편차 | 대규모 현장 배포 |

## 현장 UX 요구사항
- 큰 버튼, 고대비 UI, 장갑 모드
- 소음 환경용 음성 명령과 버튼 대체 경로
- 화면 밝기 자동 조정
- 배터리/발열 경고
- 방폭/방진/방수 등급 장비 옵션

## 디바이스 관리
- MDM 등록
- 원격 잠금/초기화
- 앱 버전 강제 업데이트
- 현장별 정책 배포
- 분실/퇴사 시 토큰 폐기
