# AI Copilot Behaviors

## AI 역할
AI는 현장 작업을 “대체”하지 않고 전문가와 작업자를 보조한다. 특히 안전·품질·규정과 관련된 결정은 전문가 또는 승인권자가 확인해야 한다.

## 주요 행동
### 1. 매뉴얼 질의응답
- 장비 모델과 오류코드에 맞는 문서 검색
- 근거 포함 답변
- 관련 도면/부품/절차 링크 표시

### 2. 다음 절차 제안
- 현재 체크리스트 단계와 대화 맥락을 기반으로 다음 확인 항목 제시
- 불확실한 경우 “확인 필요” 표시
- 위험 작업은 전문가 승인 요구

### 3. 세션 요약
- 문제 증상
- 확인한 항목
- 전문가 지시
- 수행한 조치
- 남은 이슈
- 후속 작업

### 4. 리포트 자동 생성
- 고객 제출용 간결 리포트
- 내부 기술 리포트
- 규정/감사용 리포트
- 티켓 업데이트용 구조화 JSON

### 5. 위험 경고
- 전기, 압력, 고온, 회전체, 화학물질, 방폭, 의료 개인정보 관련 경고
- 위험 문구가 감지되면 세션 화면에 경고 배너 표시
- 특정 작업은 관리자 승인 없이는 진행 불가

### 6. 지식 개선 제안
- 검색 실패 질문 모음
- 자주 쓰는 매뉴얼 chunk
- 리포트에서 새 FAQ 후보 추출
- 전문가가 수정한 AI 답변을 학습 데이터 후보로 저장

## AI가 하지 말아야 할 것
- 출처 없는 절차 지시
- 안전장치 우회 지시
- 자격 없는 사용자에게 제한 작업 안내
- 환자/고객 개인정보 노출
- 최신 승인 문서와 충돌하는 답변
- 전문가 승인 없이 고위험 조치 확정

## Human-in-the-loop 단계
| 상황 | AI 동작 | 사람 개입 |
|---|---|---|
| 일반 매뉴얼 요약 | 즉시 답변 | 선택적 검토 |
| 작업 절차 제안 | 전문가 확인 표시 | 전문가 승인 |
| 고위험 작업 | 실행 지시 차단 | Supervisor 승인 |
| 문서 충돌 | 충돌 표시 | 지식 관리자 검토 |
| 리포트 생성 | 초안 작성 | 전문가/관리자 승인 |
