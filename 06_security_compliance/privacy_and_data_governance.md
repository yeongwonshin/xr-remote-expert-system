# Privacy & Data Governance

## 데이터 분류
| 데이터 | 민감도 | 보존 정책 예시 |
|---|---|---|
| 세션 영상 | 높음 | 30~180일, 고객 설정 |
| 스냅샷 | 높음 | 리포트 첨부 시 장기 보관 |
| 음성 전사 | 중~높음 | 리포트 승인 후 원문 삭제 옵션 |
| AR 주석 로그 | 중간 | 감사 목적으로 장기 보관 |
| AI 질문/답변 | 중간 | 모델 개선 사용 여부 고객 선택 |
| 매뉴얼/도면 | 높음 | 고객 소유, 권한 기반 접근 |
| 작업 리포트 | 높음 | 규정/계약에 따라 보관 |

## 개인정보 보호
- 얼굴, 환자정보, 고객명, 차량번호 등 마스킹
- 특정 구역 촬영 금지 정책
- 리포트 생성 전 민감정보 스캔
- 외부 벤더에게는 필요한 화면/문서만 제한 제공

## 데이터 소유권
- 고객 문서, 영상, 리포트는 고객 소유
- 서비스 운영 메타데이터는 계약 범위 내 사용
- AI 모델 학습 활용은 opt-in 방식
- 데이터 삭제 요청과 내보내기 기능 제공

## 데이터 생명주기
```text
Capture → Process → Store → Review → Export → Retain → Delete/Archive
```

## Governance Role
| 역할 | 책임 |
|---|---|
| Data Owner | 보존기간과 접근정책 승인 |
| Knowledge Manager | 문서 버전과 승인 관리 |
| Security Admin | 접근권한과 감사로그 관리 |
| Compliance Officer | 규정 준수 확인 |
| AI Steward | AI 답변 품질과 실패 사례 관리 |
