# Enterprise Controls

## 관리자 정책
- 테넌트별 데이터 리전 설정
- 사용자 그룹별 기능 권한
- 외부 초대 허용/차단
- 영상 녹화 기본값 설정
- 리포트 승인 단계 설정
- AI 기능 활성화/비활성화
- 문서 보안등급별 검색 제한

## 역할 권한 예시
| 기능 | Technician | Expert | Supervisor | Admin | Vendor |
|---|---:|---:|---:|---:|---:|
| 세션 시작 | O | O | O | O | 제한 |
| AR 주석 | O | O | O | O | 제한 |
| AI 질문 | O | O | O | O | 제한 |
| 고위험 승인 | X | X | O | O | X |
| 문서 업로드 | X | △ | O | O | X |
| 리포트 승인 | X | O | O | O | X |
| 감사로그 조회 | X | X | △ | O | X |
| 외부 초대 | X | △ | O | O | X |

## 정책 엔진 조건
- 사용자 역할
- 소속 조직/현장
- 장비 보안등급
- 문서 보안등급
- 접속 위치/IP
- 디바이스 보안 상태
- 작업 위험도
- 시간대

## 컴플라이언스 준비 항목
- ISO 27001 통제 매핑
- SOC 2 보안/가용성/기밀성 통제 매핑
- 의료기관용 개인정보/촬영 제한 정책
- 산업안전 관련 승인 로그
- 항공/선박 정비 기록 보존 정책
