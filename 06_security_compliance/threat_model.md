# Threat Model

## 주요 위협
| 위협 | 설명 | 완화책 |
|---|---|---|
| 외부 벤더 과권한 | 외부 전문가가 불필요한 문서/영상 접근 | JIT 토큰, 세션 범위 제한, 감사로그 |
| 영상 데이터 유출 | 현장 영상에 민감 장비/개인정보 포함 | 암호화, 보존정책, 마스킹, 다운로드 제한 |
| AI 권한 우회 | 사용자가 AI를 통해 제한 문서 내용 요청 | 권한 필터링, 출처 통제, 감사로그 |
| 프롬프트 인젝션 | 문서나 사용자가 AI 지침을 무력화 | 시스템 프롬프트 격리, 입력 검증, 정책 필터 |
| 잘못된 AI 절차 | 근거 없는 절차로 안전사고 발생 | 출처 필수, 고위험 승인, Human-in-loop |
| 세션 탈취 | 접속 링크 노출 또는 토큰 탈취 | 짧은 만료, 디바이스 바인딩, MFA |
| 장치 분실 | 현장 디바이스 분실로 데이터 접근 | MDM, 원격 초기화, 로컬 암호화 |
| 네트워크 MITM | 영상/데이터 중간자 공격 | TLS/SRTP, 인증서 검증 |
| 로그 누락 | 사고 조사 불가 | append-only audit, 중앙 로그, 모니터링 |

## STRIDE 요약
- Spoofing: SSO/MFA/토큰 바인딩
- Tampering: 서명된 이벤트, 변경 불가 감사로그
- Repudiation: 세션 타임라인, 사용자별 액션 기록
- Information Disclosure: 암호화, 마스킹, RBAC/ABAC
- Denial of Service: autoscaling, rate limit, fallback mode
- Elevation of Privilege: 최소권한, 승인 워크플로, 정기 권한 검토

## 고위험 작업 차단 예시
- 안전 인터락 우회
- 고전압 설비 무전원 확인 없는 분해
- 압력 배출 없는 배관 분리
- 방폭 구역에서 비인증 장비 사용
- 의료 개인정보가 포함된 화면 공유
