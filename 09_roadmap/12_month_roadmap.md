# 12-Month Roadmap

## Q1: Foundation
- 모바일 앱: 영상/음성/2D 주석
- 전문가 웹 콘솔
- 세션 생성/초대/기록
- 기본 RAG 매뉴얼 검색
- 리포트 초안 생성
- 기본 RBAC와 감사로그

## Q2: Professionalization
- 스마트글래스 1종 지원
- 공간 앵커 베타
- 문서 수집/인덱싱 파이프라인
- 워크오더 API 연동
- AI 답변 평가/피드백 루프
- 관리자 콘솔 v1

## Q3: Enterprise Readiness
- SSO/OIDC/SAML
- SCIM 사용자 동기화
- 외부 벤더 JIT 접근
- 데이터 보존/마스킹 정책
- 다자 전문가 세션
- SLA/관측 대시보드

## Q4: Scale & Differentiation
- 산업별 템플릿 3종
- 유사 고장 검색
- 부품 요청/재고 연동
- 온프레미스/프라이빗 클라우드 옵션
- 다국어 전사/번역
- 고급 ROI/성과 분석

## 기술 부채 관리
- 미디어 라우터 확장성 테스트
- Vector DB 권한 필터 성능 테스트
- 세션 이벤트 스키마 안정화
- 프롬프트/가드레일 회귀 테스트 자동화
- 모바일 배터리/발열 최적화
