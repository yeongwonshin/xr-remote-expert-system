# Release Plan

## Release 0.1 — Demo Build
- 모바일 실시간 영상
- 웹 전문가 주석
- 음성 통화
- 수동 매뉴얼 업로드
- 세션 요약 데모

## Release 0.5 — Pilot Build
- 사용자/조직 관리
- 세션 녹화와 스냅샷
- RAG 검색 API
- 리포트 템플릿
- 기본 감사로그
- 파일럿 고객 1곳 운영

## Release 1.0 — Commercial Launch
- SSO
- RBAC
- 관리자 콘솔
- 워크오더 연동
- AI 리포트 승인 워크플로
- 운영 대시보드
- 고객지원 프로세스

## Release 1.5 — Enterprise
- SCIM
- 외부 벤더 JIT 접속
- 데이터 보존 정책
- 문서 권한 동기화
- 다자 전문가
- 스마트글래스 지원 확대

## Release 2.0 — Platform
- 온프레미스/프라이빗 클라우드
- 산업별 지식팩 마켓플레이스
- 유사 고장 추천
- 부품/재고 연동
- 교육 콘텐츠 자동 생성
- API/SDK 공개
